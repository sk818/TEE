# Deprecated Files

This directory contains old code, documentation, and files that are no longer used in the current TEE (TESSERA Embedding Explorer) project.

## Contents

- **Viewers**: Old 3-panel viewer (ThreePaneView.svelte), Explorer view (ExplorerView.svelte)
- **Blore Project Files**: Legacy files from the blore region analysis project
- **Old Pipelines**: Legacy data processing pipelines and scripts
- **Old Virtual Environments**: Python venv directories from earlier versions
- **Documentation**: Older planning and testing documents
- **Old Tests**: Validation and test scripts from earlier versions

## Current Active Project

The active TEE project is focused on:
1. Viewport selection via ViewportSelector.svelte
2. Downloading embeddings from TESSERA
3. Extracting and creating RGB pyramids
4. Serving pyramids via tile server
5. Viewing with the 2-panel viewer (public/viewer.html)

See root README.md for current project documentation.
